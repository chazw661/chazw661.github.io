 
  # ShowFlow Highlighter

  ShowFlow Highlighter is a Mac desktop application for live show operators.
  It connects to QLab (Figure 53) and a Google Sheets rundown document,
  automatically highlighting the current cue row in real time as the
  operator navigates the QLab playhead.
  
  ## Features

  - Real-time QLab cue tracking
  - Google Sheets rundown integration (read-only)
  - Designed for live theatre, broadcast, and live events
